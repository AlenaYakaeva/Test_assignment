# Test_assignment
Приложение для работы со счетом, которое по REST принимает два запроса GET и POST.

GET-запрос возвращает текущий баланс счета.

POST-запрос изменяет счет:

  1. Если запрос на пополнение несуществующего счета, то счет создается и сразу же пополняется.
  2. Если запрос на списание с несуществующего счета, то возвращется ошибка. Такое действие невозможно.
  3. Если запрос на списание с существующего счета, но сумма списания больше баланса, то возвращется ошибка. Такое действие невозможно.
  4. Если запрос на пополнение существующего счета или на списание с суммой, не превышающей баланс, то действие выполняется и возвращается новое состояние счета.

Приложение взаимодействует с БД PostgresQL. Вся система  поднимается с помощью docker-compose.

Переменные окружения считываются из файла config.env (файл должен быть создан вручную по примеру файла configexample.env и помещен в корневую папку приложения).
